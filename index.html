Deployment Instructions

1. GitHub: Push this `index.html` file (and any other project files) to a new GitHub repository.

2. Vercel Account: Sign up or log in to Vercel.com with your GitHub account.

3. New Project: On your Vercel dashboard, click "Add New..." → "Project".

4. Import Repository: Select your GitHub repository containing this project.

5. Configure (Optional): Vercel usually auto-detects static sites. You can typically just click "Deploy".

6. Live! Your site will be deployed to a Vercel URL. You can add a custom domain in project settings.

For each new project you want to showcase, you can create a new HTML file (e.g., `project-alpha.html`) or a subdirectory (e.g., `/project-alpha/index.html`) and link to it from the cards on this main page.
